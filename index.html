About the Encryption

This app uses the AES-GCM (Advanced Encryption Standard - Galois/Counter Mode) algorithm with a 256-bit key. AES-GCM provides both confidentiality and integrity verification. The encryption result includes the initialization vector (IV) and authentication tag for secure decryption.

Security Notice

1. Never share your encryption key with anyone.
2. This app runs entirely in your browser - no data is sent to any server.
3. For maximum security, use a password manager to generate and store keys.
4. Always verify you're using the correct key before decrypting sensitive information.
